The Goal
The goal of this website is to provide Lincoln Literacy students with a hub to which they can find materials used in class. This helps students that miss a class know what was covered so they don't fall behind. I hope that this also shows students that we care deeply about providing the best English Learning experience.
Current Status
Friday Decemeber 4th is the official release of version 1.0.0. This version makes all links functional and the about pages should be created. A mobile version of the website should also be functional although it might not look the best. The release was a bit rushed but I was too excited to not share this with the community I care so much about.
The Future
